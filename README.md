## Description

This project is the template I use for generating new clojure libs.

## Install

Since it's just me using this, it's not on clojars. To install:

```sh
$ git clone https://github.com/cldwalker/clojar-template
$ cd clojar-template
$ lein install
```

## Usage

`lein new clojar woot`
